# write - Note Taking App - MERN

## About

This project is an Google Keep Clone.

You can save your notes, categorize and archive them.

### Setup


for client

```
$ cd .../write-frontend
$ npm install
$ npm run dev

```




for server


```

$ cd .../write-backend
$ npm install
$ npm run start

```
